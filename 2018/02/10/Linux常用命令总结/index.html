Linux常用命令总结

发表于 2018-02-10 | 阅读次数:

日常操作命令

pwd 查看当前所在的工作目录

netstat -nltp 查看网络状态

文件系统操作

ls / 查看根目录下的子节点（文件夹和文件）信息

ls -l 以详细列表形式显示节点信息，别名为 ll ，加参数 a 显示隐藏文件

cd / 切换目录

mkdir a 创建目录（相对路径） 默认权限755

mkdir /data/a 创建目录（绝对路径）

mkdir -p a/b/c 创建子目录

rm -rf / 强制删除目录/文件

mv a b 修改a文件夹名称为b

touch test.1 创建文件 默认权限644

cp test.1 /app/hadoop 复制文件

cat test.1 查看文件内容（输出到控制台）

tar -cvf app.jar /home/app 打包

tar -xvf app.jar 解包

gzip app.jar 压缩

gzip -d app.jar.gz 解压缩

tar -zcvf /home/app 打包&压缩

tar -zxvf app.tar.gz (-C /目标目录) 解压&解包

使用vi编辑文件

vi用法

vi test.1 进入一般模式

输入 i 进入编辑模式

编辑完成按 Esc 键退出编辑模式，回到一般模式

输入 : 进入底行命令模式，输入 wq ，回车，保存并退出文件（或在一般模式下按 Shift + Z + Z ，也可保存并退出）

一般模式下的快捷键

a 在光标的后一位开始插入

A 在改行的最后插入

I 在该行的最前面插入

gg 直接跳到文件的首行

G 直接跳到文件的末行

dd 删除行，如果 5dd ，则一次性删除光标后的5行

yy 复制当前行, 复制多行，则 3yy，则复制当前行附近的3行

p 粘贴

v 进入字符选择模式，选择完成后，按y复制，按p粘贴

Ctrl + V 进入块选择模式，选择完成后，按y复制，按p粘贴

Shift + V 进入行选择模式，选择完成后，按y复制，按p粘贴

文件权限操作

r-4 w-2 x-1 User Group Other

chmod 755 test.1 修改权限

常用软件安装

安装JDK1.8

新建目录

mkdir apps

解压安装包

tar -zxvf jdk-8u151-linux-x64.tar.gz -C apps/

修改环境变量

vi /etc/profile

文件最后添加

export JAVA_HOME=/root/apps/jdk1.8.0_151

export PATH=$PATH:$JAVA_HOME/bin

重新加载环境变量

source /etc/profile

安装MySQL5.7

安装wget

yum -y install wget

下载MySQL源

wget https://dev.mysql.com/get/mysql57-community-release-el7-11.noarch.rpm

安装MySQL源

yum -y localinstall mysql57-community-release-el7-11.noarch.rpm

安装MySQL

yum -y install mysql-community-server

启动Mysql服务

systemctl start mysqld

设置开机启动

systemctl enable mysqld

systemctl daemon-reload

记录root本地登录临时密码

vi /var/log/mysqld.log

修改密码

mysql -u root -p

ALTER USER 'root'@'localhost' IDENTIFIED BY 'Spike07@';

设置允许远程登录

GRANT ALL PRIVILEGES ON *.* TO 'root'@'%' IDENTIFIED BY 'Spike07@' WITH GRANT OPTION;

exit;

开放3306端口

firewall-cmd --zone=public --add-port=3306/tcp --permanent

firewall-cmd --reload

配置默认编码为utf8

vi /etc/my.cnf

重启MySQL服务

systemctl restart mysqld

查看编码

show variables like '%character%';
