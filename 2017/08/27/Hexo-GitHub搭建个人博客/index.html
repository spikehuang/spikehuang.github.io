Hexo+GitHub搭建个人博客

发表于 2017-08-27 | 阅读次数:

一直以来就想搭建自己的个人博客，一来可以将自己在技术学习路上的收获分享给大家，二来记录提高自己的写作能力。之所以没有选择一些博客平台，是觉得在实际搭建的过程中能体会到动手操作的乐趣。在对比了几种搭建方法后，选定了Hexo+GitHub这种方式。

Hexo是一个快速、简单并且功能强大的博客框架。使用Hexo的前提是必须有个GitHub账号作为博客的仓库，没有GitHub的同学先注册一下。此外，搭建的过程中会涉及到Git、Node.js的一些技术。
